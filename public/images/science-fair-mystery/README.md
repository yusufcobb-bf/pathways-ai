Stage 28: Reusable theme-based images for "The Missing Project" story.

|----------|-------|------|--------|
| `gym-wide.png` | Wide gymnasium with tables | 800x500px | PNG |

## Image Generation Constraints

- Style: Simple illustration (flat, watercolor, or soft cartoon)
- NO photorealism
- NO text, logos, or branding
- NO moral judgment or depiction of wrongdoing
- Emotions: Neutral, observable through posture only
- Target file size: <100KB each

## Usage

These images are referenced in `src/data/comic-illustrations.ts` and displayed via `StoryPage.tsx`.
Fallback to gradient colors if images are missing.
